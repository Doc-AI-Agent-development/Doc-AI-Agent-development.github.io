# Doc-AI-Agent-development.github.io

Documentation site built with [Docusaurus](https://docusaurus.io/) and
deployed to GitHub Pages at <https://doc-ai-agent-development.github.io>.

## Add a document

Drop a Markdown (`.md`) file into the `docs/` folder (or a subfolder), then
commit and push to `main`. The sidebar is autogenerated, so the page appears
in the navigation automatically. See `docs/intro.md` for details.

## Local development

```bash
npm install
npm start          # dev server with live reload at http://localhost:3000
npm run build      # production build into ./build
npm run serve      # serve the production build locally
```

> Requires Node.js 20+.

## Deployment

Pushing to `main` triggers `.github/workflows/deploy.yml`, which builds the
site and publishes it to GitHub Pages. Make sure the repository's
**Settings → Pages → Build and deployment → Source** is set to
**GitHub Actions** (one-time setup).
