# 벡터 검색 툴

문서ID 범위로 하드필터를 건 뒤 의미검색하고, 청크와 출처를 반환합니다. 유일한 비결정적 경로입니다.

```
vector_search_tool/
├── SKILL.md         # 함수 호출 명세 + 하드필터 규칙
├── search.py        # 임베드 → 범위 필터 → 검색 → 청크+출처
└── embedding.py     # 임베딩 모델 설정 (미정: 3-large vs BGE-M3)
```

- **필요한 함수**: 문서ID 범위와 질의를 받아 청크와 출처를 반환하는 검색 함수.
- **SKILL.md**: 호출 명세와 하드필터 규칙(반드시 문서ID 범위로 먼저 거른 뒤 검색)을 담습니다.
- **py**: 쿼리 임베딩, 범위 필터, 의미검색, 출처를 단 반환을 담습니다. 리랭커는 자리만 두고 이후 단계에서 붙입니다.
