# 원문 인출 툴

문서ID로 원문 전체를 그대로 가져옵니다. 통째읽기 경로입니다.

```
document_fetch_tool/
├── SKILL.md       # 함수 호출 명세 + 사용 시점
└── fetch.py       # 문서ID → 원문 인출 + 길이/토큰 가드
```

- **필요한 함수**: 문서ID로 원문을 가져오는 인출 함수.
- **SKILL.md**: 호출 명세와 사용 시점(라우터가 준 문서가 골든이거나 작고 단일 주제일 때)을 담습니다.
- **py**: 저장소에서 원문 인출, 분량이 클 때의 토큰 가드를 담습니다(통째읽기 부적합 신호이며, 에이전트가 벡터 검색으로 전환합니다).
